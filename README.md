Weatherization Help Finder — NH Community Loan Fund

A lightweight, mobile-first web application that helps New Hampshire residents discover weatherization and energy assistance programs based on their household details.

Overview

This tool guides users through a short, step-by-step questionnaire and surfaces relevant assistance programs, including:

Weatherization programs
Income-based eligibility programs
Loan options
Community Action Agency (CAA) contact information

The app is fully client-side and requires no backend.

Features
Guided Questionnaire
Multi-step flow with smooth transitions
Progress tracking
Back navigation support
Eligibility-Based Results
Dynamically generated program recommendations
Categorized program cards:
Green (core programs)
Amber (secondary options)
Blue (loan programs)
Muted (informational)
Community Action Agency Finder
Dropdown-based selection
Displays:
Agency name
Phone number
Hours
Suggested call script
Mobile-First Design
Optimized for small screens (max width: 480px)
Responsive layout for larger screens
Touch-friendly UI elements
Accessibility Considerations
Semantic HTML structure
ARIA roles (role="main", aria-live="polite")
High contrast color system
Tech Stack
HTML5 – Structure
CSS3 – Styling (custom properties + responsive design)
Vanilla JavaScript – Application logic and state management
Google Fonts – Typography (Fraunces, DM Sans)

No frameworks or build tools required.
